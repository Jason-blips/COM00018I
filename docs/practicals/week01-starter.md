# Week 1 Starter Resources

This page contains all starter resources for practical work.

Resources are grouped into two folders:

- **Lab**: core starter package(s) used in practical tasks
- **Docs**: supporting documentation files

## Lab

- Open folder: [Lab resources folder](/practicals/Lab/index.html)
- Direct download: [sys3labs.zip](/practicals/Lab/sys3labs.zip)

## Docs

- Open folder: [Docs resources folder](/practicals/Docs/index.html)

## How to use these resources

1. Download and extract the lab starter package.
2. Read documentation from the `Docs` folder before Week 2.
3. Keep the same working folder and continue using it in later weeks.
